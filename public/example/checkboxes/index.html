Cases à cocher - Design System

Les cases à cocher permettent à l’utilisateur de sélectionner une ou plusieurs options dans une liste. Elles sont utilisées pour effectuer des sélections multiples (de 0 à N éléments), ou bien pour permettre un choix binaire, lorsque l’utilisateur peut sélectionner ou désélectionner une seule option.

Case à cocher seule sans erreur

<div class="rf-checkbox-group">
    <input type="checkbox" id="checkbox" name="checkbox">
    <label class="rf-label" for="checkbox">Label checkbox
    </label>
</div>

Case à cocher seule, validée

Texte de validation 

<div class="rf-checkbox-group rf-checkbox-group--valid">
    <input type="checkbox" id="checkbox-valid" name="checkbox-valid">
    <label class="rf-label rf-label--valid" for="checkbox-valid" aria-describedby="checkbox-valid-desc-valid">Label checkbox
    </label>
    <p id="checkbox-valid-desc-valid" class="rf-valid-text">
        Texte de validation
    </p>
</div>
