# @gouvfr/inputs

Les champs  permettent à un utilisateur d'entrer du contenu et données.

## Dépendances
```shell
inputs
└─ schemes
└─ core
└─ icons
└─ forms
```

## Installation
Installation via NPM:
```
npm install @gouvfr/inputs
```
Via Yarn :
```
yarn add install @gouvfr/inputs
```
