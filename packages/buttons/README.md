# @gouvfr/buttons

Le bouton est un élément d’interaction avec l’interface permettant à l’utilisateur d’effectuer une action.

## Dépendances
```shell
buttons
└─ schemes
└─ core
└─ icons
```

## Installation
Installation via NPM:
```
npm install @gouvfr/buttons
```
Via Yarn :
```
yarn add install @gouvfr/buttons
```
