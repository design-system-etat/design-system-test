# @gouvfr/selects

La liste déroulante permet à un utilisateur de choisir un élément dans une liste donnée.

## Dépendances
```shell
selects
└─ core
└─ schemes
└─ forms
```

## Installation
Installation via NPM:
```
npm install @gouvfr/selects
```
Via Yarn :
```
yarn add install @gouvfr/selects
```
