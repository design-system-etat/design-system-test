# @gouvfr/modal

La fenêtre modale aparrait par dessus le contenu entouré d'un overlay grisé qui empeche la navigation de la page

## Dépendances
```shell
modal
└─ schemes
└─ core
└─ links
└─ buttons
```

## Installation
Installation via NPM:
```
npm install @gouvfr/modal
```
Via Yarn :
```
yarn add install @gouvfr/modal
```
